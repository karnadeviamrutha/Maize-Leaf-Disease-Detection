# Maize Leaf Disease Detection using Deep Learning

## Overview
This project builds a lightweight hybrid deep learning model combining MobileNetV3 and CBAM for maize leaf disease classification.

## Features
- Image preprocessing & augmentation
- Hybrid deep learning architecture
- 98% classification accuracy
- Edge deployment optimization

## Tech Stack
Python, PyTorch, OpenCV, NumPy, Matplotlib

## Results
Achieved 98% accuracy on augmented dataset with reduced model size for real-time inference.

## How to Run
1. Clone repo
2. Install requirements
3. Run notebook
